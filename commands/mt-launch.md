Tell the user:

> The interactive thread picker is a terminal TUI — run it from your shell:
>
> ```
> mt launch
> ```
>
> Use ↑↓ to navigate, ⏎ to resume, `d` to delete, `q` to quit. The picker will resume the selected session in its original project directory.

Do not call any MCP tool — just deliver this message.
